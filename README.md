# Keypad Layout
Control window layout using Ctrl + Numeric Keypad on macOS. Press two number keys in a row while holding down Ctrl to select the new window frame.

## Examples
| `Ctrl` +     | Result           |
| :----------- | :--------------- |
| `1` then `9` | Full Screen      |
| `7` then `9` | Top Row          |
| `1` then `3` | Bottom Row       |
| `3` then `1` | Bottom Row       |
| `8` then `3` | Right Two Thirds |
